# Awesome Agentic Commerce [![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

> A curated list of awesome resources, papers, frameworks, and tools for agentic commerce - the future of AI-powered autonomous commerce systems.

Agentic commerce refers to AI-powered autonomous agents that can understand user needs, make decisions, and execute commerce-related tasks on behalf of users. This includes intelligent shopping assistants, automated procurement systems, and AI-driven sales agents.

This repository also serves as the comprehensive paper list for our survey paper on agentic commerce.
